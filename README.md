# Estafa de la tienda online falsa

# ¿Cómo funciona ésta estafa?
La estafa empieza cuando un usuario accede a las páginas web fraudulentas a través de anuncios en redes sociales, enlaces promocionales o dominios que imitan tiendas legítimas. Los delincuentes explotan la ingeniería social, aprovechando la confianza en marcas conocidas y la urgencia de ofertas como las del Black Friday. La estafa ocurre en dos puntos clave: primero, en la web clonada, que simula ser un comercio real; y segundo, en el proceso de pago, donde los datos bancarios o el dinero se transfieren a los estafadores mientras el producto nunca llega. Todo esto se puede realizar sin conocimientos avanzados de informática, utilizando plantillas web, gestores de contenido y técnicas de phishing, lo que hace que el fraude sea muy efectivo.
